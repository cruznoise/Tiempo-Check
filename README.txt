
# TiempoCheck

Sistema de consulta de uso de tiempo en internet. Proyecto web bajo arquitectura MVC con Flask y MySQL.

## Funcionalidades
- Login de usuarios (POST, AJAX)
- Dashboard con tiempo por dominio (consulta asíncrona)
- Comunicación cliente-servidor con jQuery y AJAX
- Uso de base de datos MySQL para almacenamiento

## Estructura
- `app/models/`: Modelos de Usuario y Registro
- `app/controllers/`: Lógica de rutas y funciones
- `app/templates/`: Vistas HTML (login y dashboard)
- `app/static/`: Archivos estáticos (CSS, JS, íconos)
